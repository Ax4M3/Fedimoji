**[Pour tester l'outil en ligne (GitHub Pages)](https://ax4m3.github.io/Fedimoji/)**

## Utilisation

1. Allez directement sur le [GitHub Page](https://ax4m3.github.io/Fedimoji/).
2. Entrez le nom de l'instance voulue *(par défaut [mastodon.social](mastodon.social)). Par exemple [piaille.fr](piaille.fr), [fosstodon.org](fosstodon.org), [mamot.fr](mamot.fr),...*
3. Cliquez sur **Explorer**.
4. Requête envoyée à `https://[instance]/api/v1/custom_emojis` — endpoint public, aucune authentification requise.
5. Les émojis s'affichent, vous pouvez les filtrer par catégories (fixées par l'instance) et par :shortcode:.
6. Vous pouvez copier les émojis en cliquant dessus.

## Informations

- Fedimoji est une page statique qui s'exécute entièrement dans le navigateur, aucune donnée n'est stockée ni renvoyée ailleurs.
- Trois catégories sont présentes par défaut : nombre d'émojis, nombre de catégories et nombre d'émojis animés.
- Les émojis copiés ne sont utilisables uniquement dans les instances sélectionnées.
